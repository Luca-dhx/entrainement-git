A propos

Cette page sert de support pour s'entrainer avec Git : commits, branches, merges, et plus encore.

Commandes utiles

  • git init
  • git add .
  • git commit -m "message"
  • git branch nom-branche
  • git checkout nom-branche
  • git merge nom-branche
  • git log --oneline
